## Assignment Name 
Assignment #3 - DQN (Deep Q-Learning or Deep Q Network)

## Assignment Synopsis
Implement the DQN algorithms on the Pong environment from OpenAI Gymnasium Documentation (https://www.gymlibrary.dev/environments/atari/pong/). Train an agent to play Pong efficiently.

## Files Description
1. assignment3_utils.py: This is an assist in interacting with the Pong environment. Provide a few methods for pre-processing of the image, including cropping, switching to grayscale, and downsizing
2. Pong_DQL.ipynb: Implement the DQN algorithms on the Pong environment
3. requirements.txt: Library version requirements
